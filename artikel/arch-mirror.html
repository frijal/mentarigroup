Skrip arch-mirror.sh: Pilih Mirror Tercepat

Skrip arch-mirror.sh ini menggunakan utilitas reflector untuk secara otomatis memilih mirror tercepat untuk sistem Arch Linux Anda. Proses ini penting untuk mempercepat unduhan paket dan pembaruan sistem.

⚠️ Jalankan skrip ini dengan hak akses sudo karena akan mengubah file konfigurasi sistem di /etc/pacman.d/mirrorlist.

Kode Skrip

#!/bin/bash
# arch-mirror.sh - Pilih mirror Arch Linux tercepat secara otomatis

set -e

echo "[INFO] Memastikan 'reflector' terinstal..."
sudo pacman -Sy --noconfirm reflector

echo "[INFO] Mem-backup mirrorlist yang ada..."
sudo cp /etc/pacman.d/mirrorlist /etc/pacman.d/mirrorlist.bak

echo "[INFO] Mencari 10 mirror terbaru dan tercepat..."
sudo reflector --latest 10 --sort rate --save /etc/pacman.d/mirrorlist

echo "[INFO] Menyinkronkan ulang database paket..."
sudo pacman -Syy

echo "[SELESAI] Mirrorlist telah diperbarui dengan server tercepat!"

Bagikan Skrip Ini
