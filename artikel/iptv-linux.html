📺 Linux dan Internet Protocol Television (IPTV)

📌 Sumber asli: Linux.org

🔎 Apa itu IPTV?

IPTV (Internet Protocol Television) itu teknologi yang bikin kita bisa nonton TV, film, atau serial lewat internet—tanpa perlu antena atau parabola. Bedanya sama Netflix/Roku, IPTV biasanya buat siaran langsung (live stream), bukan video on-demand.

🖥️ Aplikasi IPTV di Linux

Dahulu ada FreeTuxTV, tapi sekarang lebih populer pakai Hypnotix (bawaan Linux Mint, tapi bisa kok di Ubuntu/distro lain). Cara pakainya gampang, tampilannya juga kekinian.

⚙️ Instalasi Hypnotix di Ubuntu

Install Hypnotix paling gampang lewat PPA supaya dependensi aman:

  1. sudo apt install software-properties-common -y
  2. sudo add-apt-repository ppa:kelebek333/mint-tools
  3. sudo apt update
  4. sudo apt install hypnotix -y

Buka Hypnotix, cek menu ☰ → About buat lihat versi.

📂 Kategori Konten

Di halaman utama Hypnotix ada tiga kategori: TV Channels, Movies, dan Series. Jumlah channel bisa beda-beda, ada yang 24 jam, ada yang kadang off.

➕ Menambahkan Channel Baru

  1. Unduh file M3U dari IPTV-org.
  2. Simpan di ~/.hypnotix/
  3. Di Hypnotix: menu Providers → Add a new provider…
  4. Pilih Local M3U File atau M3U URL
  5. Isi nama dan path/URL

⭐ Fitur Tambahan Hypnotix

  • Favorites: klik ikon bintang buat simpan channel favorit
  • Keyboard Shortcuts:
    • F11 → Fullscreen
    • F1 → Tentang Hypnotix
    • F2 → Info audio/video
    • CTRL+F → Cari channel
    • CTRL+K → Daftar shortcut
    • CTRL+R → Reload provider
    • CTRL+Q → Keluar

🎯 Kesimpulan

Dengan Hypnotix atau aplikasi sejenis, Linux jadi platform IPTV yang praktis. Kamu bisa:

  • Nonton siaran TV internasional
  • Tambah database channel sendiri
  • Simpan favorit biar akses lebih cepat
  • Bahkan beberapa aplikasi bisa rekam siaran untuk ditonton offline
