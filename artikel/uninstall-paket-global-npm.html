🗑️ Cara Gampang Uninstall Paket Global Node.js (npm, yarn, pnpm)

Udah install banyak tool tapi bingung cara hapusnya? Ini panduan santai buat uninstall paket global di semua OS (Linux, macOS, dan Windows).

Dipublikasikan: 27 Okt 2025 npm yarn pnpm Uninstall

Gunakan flag -g untuk menghapus paket global.

Di artikel sebelumnya, kita udah bahas cara melihat daftar semua paket global yang terpasang. Nah, setelah kamu lihat daftarnya, pasti ada beberapa paket yang udah nggak kepake lagi, 'kan? Entah itu tool lama, salah install, atau cuma menuh-menuhin tempat.

Gampang aja, cara hapusnya mirip banget sama cara install-nya. Kamu cuma perlu pakai perintah *uninstall* dari package manager yang kamu gunakan.

1. Terminal (Linux & macOS) 🐧🍎

Perintah di terminal Linux (seperti Ubuntu, Fedora, BlankOn, CachyOS, dll.) dan macOS itu pada dasarnya sama persis.

Untuk NPM:

Gunakan perintah npm uninstall dengan flag -g (singkatan dari --global).

npm uninstall -g <nama-paket>

Untuk Yarn:

Yarn pakai perintah yarn global remove. Lebih deskriptif, ya?

yarn global remove <nama-paket>

Untuk PNPM:

PNPM mirip NPM, pakai pnpm remove (atau rm) plus flag -g.

pnpm remove -g <nama-paket>
Catatan Penting (Linux/macOS):
Kalau dulu kamu install-nya pakai sudo (misal: sudo npm install -g ...), maka kamu juga harus pakai sudo buat uninstall-nya. Contoh: sudo npm uninstall -g <nama-paket>.

2. Windows (PowerShell) 💻

Kabar baik buat pengguna Windows! Perintahnya tetap sama persis kok di dalam PowerShell.

Untuk NPM:

npm uninstall -g <nama-paket>

Untuk Yarn:

yarn global remove <nama-paket>

Untuk PNPM:

pnpm remove -g <nama-paket>
Catatan Penting (Windows):
Paket global itu seringnya di-install di folder yang dilindungi sistem (kayak C:\Program Files\nodejs). Jadi, kamu mungkin perlu menjalankan PowerShell sebagai Administrator (klik kanan ikon PowerShell > "Run as administrator") biar proses hapusnya lancar jaya.

3. Contoh Praktis (Studi Kasus) 🧑‍💻

Misalnya, kamu baru aja jalanin perintah dari artikel sebelumnya:

npm list -g --depth=0

Terus kamu lihat ada paket nodemon dan pm2 yang udah nggak kamu pakai lagi.

Yaudah, tinggal eksekusi aja perintah uninstall-nya satu per satu:

npm uninstall -g nodemon
npm uninstall -g pm2

Selesai! Paket-paket itu sekarang udah bersih dari laptop kamu.
