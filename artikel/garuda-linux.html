Garuda Linux Rilis ISO Baru: September 2025

Garuda Linux, distro berbasis Arch yang dikenal dengan tampilan visual memukau, merilis ISO snapshot terbaru. Ini adalah pembaruan kumulatif yang mencakup semua paket terbaru dan perbaikan bug penting.

Perubahan Utama

Meskipun bukan rilis mayor, snapshot ini membawa beberapa perbaikan krusial:

  • Mesa 25.2.2: Memperbaiki bug logout acak yang mengganggu sesi Plasma Wayland.
  • Peningkatan Deteksi Hardware: Alat garuda-hardware-tool kini lebih baik dalam mendeteksi GPU.
  • Stabilitas Update: Perintah udevadm trigger dinonaktifkan sementara untuk mencegah sistem crash saat pembaruan systemd/udev.

Pembaruan Edisi Dr460nized

Edisi andalan Garuda, Dr460nized, juga mendapatkan pembaruan, termasuk Plasma 6.4.4, shell Fish 4.0, dan kernel Linux 6.16 terbaru.

Cara Update

Jika Anda sudah menggunakan Garuda Linux, Anda tidak perlu melakukan instalasi ulang. Cukup jalankan perintah berikut di terminal untuk mendapatkan semua pembaruan terbaru:

garuda-update
