Panduan Perintah du di Linux: Cek Pemakaian Ruang Disk

Di Linux, untuk mengetahui berapa banyak ruang yang dipakai file atau folder, kita tidak bisa sekadar klik kanan. Untungnya ada du (disk usage), tool ajaib untuk memeriksa ukuran file/direktori lewat command line.

Opsi Paling Populer

Berikut adalah beberapa kombinasi opsi du yang paling sering digunakan:

  • -h → Human-readable, menampilkan ukuran dalam format yang mudah dibaca (KB, MB, GB).
  • -s → Summarize, hanya menampilkan total ukuran, tanpa rincian subfolder.
  • -a → All, menampilkan ukuran setiap file individual selain direktori.

Contoh Penggunaan Praktis

Cek Ukuran Total Direktori Saat Ini

du -sh .

Cek Ukuran Setiap Folder di Dalamnya

du -sh *

Keterbatasan: Masalah Hard Link

Perintah du standar dapat memberikan hasil yang kurang akurat jika sistem file Anda mengandung hard link. Hard link memungkinkan satu file untuk muncul di banyak lokasi tanpa memakan ruang ekstra. Akibatnya, du bisa menghitung ukuran file yang sama berulang kali.

Solusi dengan Skrip Perl

Untuk mendapatkan hasil yang lebih presisi, Anda bisa menggunakan skrip Perl berikut. Skrip ini melacak nomor inode setiap file untuk memastikan hard link tidak dihitung ganda.

#!/usr/bin/perl -w
use strict;
my %hash;
my $total_blocks = 0;
my @todo = @ARGV; push(@todo, '.') unless @todo;
foreach my $fn (@todo) {
  my($inode,$nlinks,$blocks) = (lstat($fn))[1,3,12];
  if (-d _) {
    opendir(DP, $fn) or die $!;
    push(@todo, map("$fn/$_", grep(!/^\.\.?$/, readdir(DP))));
    closedir DP;
  } elsif ($nlinks > 1) {
    if (!defined($hash{$inode})) {
      $hash{$inode} = $nlinks - 1;
      next;
    }
    next if --$hash{$inode};
  }
  $total_blocks += $blocks;
}
print "$total_blocks blocks\n";
print int($total_blocks / 2 + .5), "K\n";

Simpan skrip di atas sebagai file (misalnya `du_accurate.pl`), berikan izin eksekusi (`chmod +x du_accurate.pl`), lalu jalankan di direktori target. Hasilnya akan jauh lebih akurat.
