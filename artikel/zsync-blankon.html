Panduan ZSync untuk ISO BlankOn Rote

Hemat Kuota & Efisien dalam Mengunduh ISO BlankOn Terbaru

Pendahuluan

Bagi Anda yang hobi mencoba ISO terbaru dari distribusi Linux BlankOn, artikel ini akan membahas cara menggunakan ZSync untuk mengunduh ISO BlankOn Rote dengan efisien. BlankOn Rote dikenal rajin merilis CD harian (*daily build*) dan saat ini sudah mencapai versi Jahitan Lima.

Artikel ini ditujukan bagi para Pendekar BlankOn agar dapat menyediakan versi .iso.zsync untuk setiap pembaruan, sehingga pengguna dengan kuota terbatas tetap bisa ikut menguji.

Apa Itu ZSync?

ZSync adalah alat yang memungkinkan Anda untuk mengunduh hanya bagian yang berubah pada file ISO, bukan mengunduh ulang seluruh file yang berukuran besar (rata-rata 700MB).

Manfaat ZSync:

  • Hemat kuota internet
  • Proses unduhan lebih cepat
  • Mudah digunakan

Cara Menggunakan ZSync

  1. Install Paket ZSync
    Pastikan paket zsync sudah terinstall. Jika belum, jalankan perintah: 
    sudo apt install zsync
  2. Ganti Nama File ISO Lama
    Ubah nama file ISO lama Anda agar sesuai dengan nama file di server. Contoh: 
    mv BlankOn-8.0-Jahitan-5-desktop-amd64.iso rote-desktop-amd64.iso
  3. Jalankan ZSync
    Jalankan perintah ZSync dari terminal dengan menunjuk ke alamat ISO terbaru yang memiliki ekstensi .zsync: 
    zsync http://cdimage.blankonlinux.or.id/blankon/livecd-harian/20120502/rote-desktop-amd64.iso.zsync
    Atau, jika ingin menunjuk file ISO lama secara spesifik: 
    zsync -i BlankOn-8.0-Jahitan-5-desktop-amd64.iso http://URL_SERVER/NAMA_FILE.iso.zsync

Kesimpulan

Dengan menggunakan ZSync, Anda dapat menghemat kuota internet dan waktu dalam mengunduh ISO BlankOn Rote terbaru. Semoga panduan ini bermanfaat bagi seluruh Pendekar BlankOn!
