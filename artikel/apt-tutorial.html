Tutorial APT (Advanced Package Tool)

APT adalah manajer paket bawaan untuk sistem operasi Debian dan turunannya (seperti Ubuntu). Dengan APT, Anda bisa dengan mudah mencari, menginstal, menghapus, dan mengelola aplikasi langsung dari terminal.

1. Memperbarui Sistem

Selalu mulai dengan memperbarui daftar paket Anda untuk memastikan Anda mendapatkan versi terbaru.

sudo apt update && sudo apt upgrade -y

2. Mencari dan Menginstal Aplikasi

Gunakan apt search untuk menemukan nama paket dan apt install untuk memasangnya.

# Mencari aplikasi (contoh: htop)
apt search htop

# Menginstal aplikasi
sudo apt install htop -y

3. Menghapus Aplikasi dan Membersihkan Sistem

Gunakan apt remove untuk menghapus aplikasi dan autoremove serta clean untuk membersihkan file sisa.

# Menghapus aplikasi
sudo apt remove htop -y

# Menghapus paket yang tidak lagi diperlukan
sudo apt autoremove -y

# Membersihkan cache paket
sudo apt clean
