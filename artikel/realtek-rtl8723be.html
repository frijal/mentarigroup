Perbaiki Tangkapan Wi-Fi Realtek RTL8723BE di Ubuntu Linux

Jika sinyal Wi-Fi di laptop Anda yang menggunakan chip Realtek RTL8723BE terasa lemah di Linux, jangan khawatir. Solusinya cukup mudah: kita akan mengunduh dan menginstal driver yang sudah diperbaiki dari GitHub.

Langkah Cepat Perbaikan

Intinya adalah kita akan meng-clone repositori driver dari GitHub, lalu menjalankan skrip instalasi yang sudah disediakan. Proses ini akan mengompilasi dan memasang modul kernel yang tepat untuk perangkat Anda.

1. Siapkan Alat Build

Pertama, pastikan semua paket yang diperlukan untuk kompilasi sudah terinstal.

sudo apt update && sudo apt install -y build-essential git dkms linux-headers-$(uname -r)

2. Unduh dan Instal Driver

Selanjutnya, unduh repositori driver, masuk ke direktorinya, dan jalankan skrip instalasi.

git clone https://github.com/tarunbatra/fixRTL8723BE
cd fixRTL8723BE
chmod +x install.sh
sudo ./install.sh

3. Reboot

Setelah instalasi selesai, reboot komputer Anda agar modul kernel yang baru dapat dimuat.

sudo reboot

Setelah restart, periksa kembali koneksi Wi-Fi Anda. Seharusnya sinyal yang diterima kini jauh lebih kuat dan stabil.

Tips: Siapkan koneksi internet alternatif (misalnya, tethering dari ponsel atau kabel Ethernet) untuk berjaga-jaga selama proses instalasi driver ini.
