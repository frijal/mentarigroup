Ngoprek Conky: Konfigurasi Bahasa Indonesia & Autostart

Conky adalah system monitor ringan untuk desktop Linux yang menampilkan berbagai statistik langsung di layar Anda. Panduan ini akan memandu Anda mulai dari instalasi, membuat skrip awal, hingga mengatur autostart dengan konfigurasi berbahasa Indonesia.

Langkah 1: Instalasi Conky

Gunakan manajer paket distro Anda untuk menginstal Conky. Berikut beberapa contohnya:

# Untuk Debian/Ubuntu
sudo apt-get install conky-all

# Untuk Fedora
sudo dnf install conky

# Untuk Arch Linux
sudo pacman -S conky

Langkah 2: Buat Skrip Awal

Buat file bernama ~/.mulaiconky.sh. Skrip ini memastikan Conky menunggu desktop siap sebelum berjalan.

#!/bin/bash
sleep 10 && conky -c "$HOME/.conkyrc" &

Jangan lupa berikan hak eksekusi:

chmod +x ~/.mulaiconky.sh

Langkah 3: Atur Autostart

Buat file .desktop di ~/.config/autostart/ agar Conky berjalan otomatis setiap kali Anda login.

mkdir -p ~/.config/autostart
cat > ~/.config/autostart/conky.desktop <<'EOF'
[Desktop Entry]
Type=Application
Name=Conky
Exec=sh -c "sh ~/.mulaiconky.sh"
StartupNotify=false
Terminal=false
EOF

Langkah 4: Konfigurasi .conkyrc

Simpan konfigurasi berikut sebagai ~/.conkyrc. Konfigurasi ini sudah dalam Bahasa Indonesia dan mencakup pemantauan CPU, memori, disk, dan jaringan.

conky.config = {
    update_interval = 1,
    double_buffer = true,
    own_window = true,
    own_window_type = 'dock',
    own_window_argb_visual = true,
    own_window_argb_value = 140,
    own_window_hints = 'undecorated,sticky,skip_taskbar,skip_pager,below',
    alignment = 'top_right',
    gap_x = 30,
    gap_y = 50,
    minimum_width = 280,
    use_xft = true,
    font = 'DejaVu Sans:size=10',
    default_color = 'white',
    color1 = '#22d3ee',
};

conky.text = [[
${color1}SISTEM ${hr 2}
${color}Host: ${nodename} | Kernel: ${kernel}
Uptime: ${uptime} | Beban: ${loadavg}

${color1}CPU ${hr 2}
${cpugraph 20,260}
Inti 1: ${cpu cpu1}% | Inti 2: ${cpu cpu2}%

${color1}MEMORI ${hr 2}
RAM: ${mem} / ${memmax} ${alignr}${memperc}%
${membar 4,260}

${color1}JARINGAN ${hr 2}
Wi-Fi (wlan0): Up: ${upspeed wlan0} | Down: ${downspeed wlan0}
IP Lokal: ${addr wlan0}
]];

Sesuaikan nama interface jaringan (misalnya wlan0, eth0) dengan yang ada di sistem Anda. Anda bisa memeriksanya dengan perintah ip a.
