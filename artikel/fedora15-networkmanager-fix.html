Fedora 15 NetworkManager Fix Modem USB/Ponsel

Kalem, modem USB macet habis update? Kita betulin bareng

Ada bug di paket NetworkManager tanggal 26 Mei 2011 yang bikin koneksi seluler gagal. Solusinya: pasang build perbaikan, bersihkan koneksi lama, dan start fresh.

Bug: gnome-0.8.999.3.git20110526.fc15 • Fix: NetworkManager-gnome-0.8.9997-1.git20110531.fc15

Garis besarnya: Unduh paket RPM yang dibutuhin, install bareng via terminal, hapus profile lama di system-connections, lalu jalankan NetworkManager lagi. Selesai, modem USB dan tethering ponsel balik waras.

Peringatan singkat: Sebelum menghapus isi /etc/NetworkManager/system-connections, backup dulu foldernya. Contoh: sudo cp -a /etc/NetworkManager/system-connections{,.backup}

Langkah ringkas via Terminal

Kita pakai terminal biar rapi dan bisa ditelusuri. Pastikan koneksi internet aktif (pakai kabel/Wi‑Fi). Simpan semua RPM di satu folder.

1. Unduh paket yang diperlukan

mkdir -p ~/Downloads/nm-fix && cd ~/Downloads/nm-fix# Dependensi (contoh i686 sesuai sumber lama)# dbus, glib2, dbus-glib devel# Paket NetworkManager versi 0.8.9997-1.git20110531.fc15# (gunakan tautan asli yang kamu punya/arsip lokal)

Daftar tautan historis (arsip):

  • dbus-devel-1.4.6-4.fc15.i686.rpm
  • glib2-devel-2.28.6-2.fc15.i686.rpm
  • dbus-glib-devel-0.92-2.fc15.i686.rpm
  • NetworkManager-0.8.9997
  • NetworkManager-glib-0.8.9997
  • NetworkManager-devel-0.8.9997
  • NetworkManager-gnome-0.8.9997
  • NetworkManager-glib-devel-0.8.9997
  • NetworkManager-debuginfo-0.8.9997

2. Instal massal RPM lokal

cd ~/Downloads/nm-fix
sudo yum localinstall *.rpm

3. Bersihkan profil koneksi lama (backup dulu!)

# Backup profil koneksi
sudo cp -a /etc/NetworkManager/system-connections{,.backup}
# Hapus profil lama (akan dibuat ulang)
cd /etc/NetworkManager/system-connections/
sudo rm -f *

4. Mulai ulang NetworkManager

# Opsi A (service)
sudo service NetworkManager restart
# Opsi B (systemd, kalau ada)
sudo systemctl restart NetworkManager
# Opsi C (jalanin biner langsung - tidak selalu diperlukan)
# sudo NetworkManager

Hasil yang diharapkan: Applet NM muncul normal, modem USB/ponsel terdeteksi, kamu bisa buat koneksi baru, dan internet jalan lagi. Kalau tetap gagal, restore profil: sudo rm -rf /etc/NetworkManager/system-connections/* &&sudo cp -a /etc/NetworkManager/system-connections.backup/*/etc/NetworkManager/system-connections/ lalu restart NM.
