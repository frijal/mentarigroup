SL di Terminal: Kereta Api Nyelonong Gara-gara Salah Ketik

Sepoor, kreto, sepur, kereta api - semua sama. Dan kali ini, dia lewat di terminal karena kamu salah ketik ls jadi sl. Serius.

21 Sep 2025 • Linux Fun

Kisah bermula dari salah ketik

Pengen lihat isi direktori, niatnya ngetik ls. Eh jari kebablasan jadi sl. Muncul pesan: 

The program 'sl' is currently not installed.
You can install it by typing: sudo apt-get install sl

Karena penasaran, ya sudah… dipasang saja. Dan begitu ketik sl, jreng - kereta api ASCII melintas gagah di terminal!

Apa itu SL?

SL adalah program kecil nan legendaris yang menampilkan animasi kereta api ASCII di terminal. Didesain untuk “menghibur” (dan sedikit mengejek) kita yang sering salah ketik ls.

Instal cepat
sudo apt-get update && sudo apt-get install -y sl
Jalankan
sl
Opsi lucu
sl -a (mengerikan), sl -l (kereta panjang), sl -F (kereta terbang - iya, serius)
