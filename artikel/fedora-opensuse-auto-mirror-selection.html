Otomatis Memilih Mirror Linux Tercepat

Untuk mempercepat proses update dan instalasi paket di Linux, kita bisa memilih mirror repository tercepat. Berikut adalah cara otomatis untuk melakukannya di berbagai distro populer.

ðŸ“Œ Catatan: Jalankan perintah di bawah dengan sudo agar dapat mengubah file konfigurasi sistem.

1. Debian & Ubuntu (netselect-apt)

# Instal netselect-apt jika belum ada
sudo apt update && sudo apt install -y netselect-apt

# Jalankan untuk menghasilkan sources.list baru
sudo netselect-apt stable

# Ganti sources.list lama dengan yang baru (otomatis di-backup)
sudo mv sources.list /etc/apt/sources.list

2. Arch Linux (reflector)

# Pilih 5 mirror terbaru dari Indonesia dan urutkan berdasarkan kecepatan
sudo reflector --country "Indonesia" --latest 5 --sort rate --save /etc/pacman.d/mirrorlist

# Sinkronkan ulang database paket
sudo pacman -Syyu

3. Fedora (DNF)

Fedora secara otomatis memilih mirror terbaik. Untuk memaksa pembaruan, bersihkan cache dan refresh.

sudo dnf clean all
sudo dnf makecache --refresh

4. openSUSE (Zypper)

Sama seperti Fedora, openSUSE juga otomatis. Anda bisa memaksa refresh dengan perintah berikut.

sudo zypper refresh
