Drauger OS: Distro Linux Khusus Gaming

Drauger OS adalah distro Linux berbasis Ubuntu yang dirancang khusus untuk gaming. Fokus utamanya adalah performa, sehingga aplikasi yang tidak esensial dihilangkan agar sistem tetap ringan dan responsif.

Spesifikasi Sistem

Meskipun ringan, Drauger OS tetap memerlukan spesifikasi yang mumpuni untuk pengalaman gaming yang optimal.

  • Minimum: CPU Dual-core 1.8 GHz, 1 GB RAM, 32 GB Storage.
  • Rekomendasi: CPU Quad-core 2 GHz, 6 GB DDR4 RAM, 128 GB NVMe SSD, dan GPU setara Nvidia GTX 950 / AMD RX 450.

Proses Instalasi

Proses instalasi Drauger OS cukup standar. Setelah boot dari USB, Anda bisa memilih "Try" untuk Live Session atau langsung "Install". Installer menyediakan beberapa mode, termasuk "Normal Installation" untuk pengaturan manual.

Gaming di Drauger OS

Drauger OS sudah siap pakai untuk gaming dengan berbagai platform populer yang terinstal secara default:

  • Steam
  • GameHub
  • Heroic Games Launcher (untuk Epic Games Store)
  • Lutris

Tips: Sebelum menggunakan Lutris, pastikan driver GPU Anda sudah sesuai. Jika diperlukan, install paket meta-vulkan-drivers untuk arsitektur i386.

Catatan Penting

  • Jangan melakukan pembaruan sistem sembarangan, karena dapat menyebabkan error saat boot.
  • Upgrade antar versi besar (misalnya dari 7.6 ke 7.7) memerlukan instalasi ulang.

Kesimpulan

Jika Anda mencari distro Linux yang fokus pada gaming dengan performa optimal, Drauger OS adalah pilihan yang sangat layak untuk dicoba. Coba dulu di Virtual Machine, dan jika Anda puas, install di PC atau laptop gaming Anda.
